# Plane Task ID 与 Commit 命名

已迁至 [da-workflow/task-naming](../../da-workflow/references/task-naming.md)（权威源）。

速查：

| 对象                   | 分隔符     | 示例                       |
| ---------------------- | ---------- | -------------------------- |
| Module（钉表 depth-1） | 短横线 `-` | `S3.3-功能开发`            |
| Issue（任务）          | 中点 `·`   | `S3.3.15 · 预警页筛选对接` |

禁止用 `·` 另建同代号 Module（会与钉表重复）。细则见 [plane-dingtalk-module-rules](../../da-workflow/references/plane-dingtalk-module-rules.md)。
